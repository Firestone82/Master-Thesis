\subsection{Struktura modulu}
\label{subsec:implementation-module}

LLM modul je uspořádán jako standardní Django aplikace s osmi hlavními soubory a několika soubory v API, přičemž každý z nich pokrývá jasně ohraničenou část funkcionality.
Toto rozdělení umožňuje oddělit logiku komunikace s jazykovým modelem od správy promptů, definice výstupního schématu a ukládání výsledků.
